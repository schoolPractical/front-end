# 使用手册

1. 将文件git下来
2. 打开命令行控制台 进入该文件
3. 执行 `npm install` 安装依赖包
4. 执行 `npm run dev` 启动
5. 打开 http://localhost:8080 进入页面



## 文件说明

> 所有内容在 `src文件夹` 里操作

`./src/api/ ` 里面存放的是交互用的内容， 等要做接口后再弄

`./src/assets/` 这个文件夹用来存放图片

`./src/components/` 这个文件夹存放公共组件

`./src/router/` 存放路由

`./src/view/` 存放具体的内容
